# 智能药箱

![](https://img.shields.io/badge/language-python-blue)![](https://img.shields.io/badge/version-v1.0.0-success)![](https://img.shields.io/badge/license-MIT-ff63b4)

## 主要功能

- 药品查询(实时爬取药品信息)
- 药品管理(药品的增删改查)
- 远程监控(远程查看药箱的实时情况)
- 远程控制(使用小程序可操作药箱)
- 智能提醒吃药
- 用户管理
