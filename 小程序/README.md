# 小程序部分

## 安装

下载`微信开发者工具`并导入demo文件夹。

静等项目编译完成即可使用

> 小程序内的API截止到发布为止依旧有效，不定时会将服务器关闭。有需要可以到后端的界面部署自己的后端
