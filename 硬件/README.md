# 硬件部分

硬件使用了arduino作为驱动的主板，物联网平台使用了OneNET平台



> 提供了仿真和源码两个部分
>
> 如果你有arduino可以直接烧写main.c
>
> 如果没有真机可以下载proteus进行仿真，仿真文件是除main.c之外的其他部分
